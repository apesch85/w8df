Morse code

Useful whether you know code or not.

This part of the site helps amateurs learn Morse code. SMARS believes Morse code is valuable, while also recognizing that not knowing code does not make anyone less of an operator.

Software

A CW Trainer using the Koch method is referenced by the original site. Version 4 added WAV recording, remembered session settings, simplified Farnsworth controls, user-definable Morse characters, and rig control using RTS or DTR for over-the-air training.

The original site links a ZIP download / setup package.

W1AW schedule

Scheduled transmissions.

W1AW transmits on-the-air code practice and bulletins of interest to hams. Bulletins are also available by email and on the web.

W1AW’s schedule is at the same local time throughout the year. UTC is Eastern Time +4 hours during daylight time and +5 hours during standard time.
